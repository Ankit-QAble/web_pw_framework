# About

This framework is a comprehensive solution for web automation using [Playwright](https://playwright.dev/) and TypeScript.

## Features

- **Page Object Model**: Structured and maintainable test code.
- **Data-Driven Testing**: Support for JSON, CSV, and Excel data sources.
- **Visual Regression**: Pixel-perfect visual comparisons.
- **Reporting**: Detailed HTML reports and email notifications.
- **CI/CD Integration**: Ready-to-use configurations for Docker, GitHub Actions, and Azure DevOps.
